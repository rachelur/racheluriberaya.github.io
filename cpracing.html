TriFold Brochure

In my Digital Printing and Emerging Technologies in Graphic Communication class we learned about integrate DataMerge automation. I decided to create a tri fold brochure for my club, Cal Poly Racing. The purpose of this project was to use a simple and clean design that will appeal to different audiences. We learned how to design a database that would allow us to personalize each brochure. I've always wondered how companies send out brochures and emails, but were also able to customize them to a certain level for each indiviual. It was neat to learn how it works using Excel and InDesign.

Some of my roles included:

  1. Photoshoped images.
  2. Created final visual designs.
